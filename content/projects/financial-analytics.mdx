## Overview

A personal finance analytics platform built around **dividend-growth investing**, tracking two complementary ETFs:

- **SCHD** (Schwab US Dividend Equity) — dividend growth focus, ~3.5% yield, ~10% DGR
- **JEPQ** (JPMorgan Nasdaq Equity Premium Income) — covered-call income, ~9–11% yield

## Data Ingestion Pipeline

```python
import yfinance as yf
import pandas as pd
from sqlalchemy import create_engine

TICKERS = ['SCHD', 'JEPQ', 'VTI', 'BND']

def fetch_dividends(ticker: str) -> pd.DataFrame:
    stock = yf.Ticker(ticker)
    divs  = stock.dividends.reset_index()
    divs.columns = ['date', 'dividend']
    divs['ticker']    = ticker
    # Trailing 12-month yield
    close_avg         = stock.history(period='1y')['Close'].mean()
    divs['yield_ttm'] = divs['dividend'].rolling(4).sum() / close_avg
    return divs

engine = create_engine('sqlite:///portfolio.db')
for ticker in TICKERS:
    fetch_dividends(ticker).to_sql(
        f'dividends_{ticker.lower()}', engine,
        if_exists='replace', index=False
    )
```

## Key Metrics

### Yield on Cost

$$YoC = \frac{D_{annual}}{P_{cost}} \times 100\%$$

### Dividend Growth Rate over $n$ years

$$DGR_n = \left(\frac{D_{current}}{D_{n}}\right)^{\frac{1}{n}} - 1$$

## Monte Carlo Income Projection

```python
import numpy as np

def monte_carlo_income(
    initial: float,
    yield_rate: float,
    growth_rate: float,
    years: int = 20,
    n: int = 10_000,
) -> np.ndarray:
    totals = []
    for _ in range(n):
        shock  = np.random.normal(0, 0.02, years)   # ±2% annual variance
        income = sum(
            initial * yield_rate * (1 + growth_rate + shock[y]) ** y
            for y in range(years)
        )
        totals.append(income)
    return np.array(totals)

sim = monte_carlo_income(50_000, 0.035, 0.10)
print(f"Median 20-yr income : ${np.median(sim):>12,.0f}")
print(f"95th percentile     : ${np.percentile(sim, 95):>12,.0f}")
```

## Rebalancing Logic

An alert fires when any holding drifts beyond ±5% of its target weight:

$$\Delta_{drift} = |w_{actual} - w_{target}| > 0.05$$
