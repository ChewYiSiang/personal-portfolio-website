# Capstone Project: 🌱 Garden AI Assistant

An AI agent application using **Google AI Agent Developer Kit (ADK)**.
This Capstone Project is part of the 5-Day AI Agents Intensive Course with Google (Nov 10 - 14, 2025). The project focuses on applying large language models, autonomous AI agents using Google's Gemini API and modern agentic frameworks, and tool integrations to build end-to-end autonomous AI systems.

## Problem
Community gardens frequently struggle with communication, inconsistent access to expertise, and difficulties in managing shared resources. Individual gardeners may encounter uncertainty when diagnosing plant health issues, identifying effective organic pest solutions, or planning out seasonal activities. These overlapping needs can lead to inefficiencies, repeated mistakes, or missed opportunities for collaboration and learning. By giving gardeners assistance with gardening focused response, it can help them plan out their gardening journey without having to search through forums, websites or having to look around for the availability of their community garden's resources.

## Solution
A multi-agent garden assistant with a conversational AI system that helps community gardeners optimize their growing practices through coordinated and actionable assistance.

Agents are ideal for this project because the challenges faced by community gardeners are both diverse and highly contextual: plant care, resource coordination, and knowledge sharing all require distinct expertise and adaptive decision-making. Multi-agent systems make it possible to modularize these capabilities. Each specialized agent, whether focused on plant health, inventory, or scheduling, can process domain-specific queries and collaborate with each other, resulting in more accurate and personalized assistance. This structure allows the AI to handle complex workflows that go beyond simple question answering, such as diagnosing plant problems and then proactively recommending suitable treatments or finding the necessary tools, mirroring the collaborative dynamics found in real gardening communities.

## Features

✅ **LLM Agent** - Using `LlmAgent` with Gemini model.  
✅ **Multi-agent system** - Using workflow agents like parallel agent, sequential agent and loop agent to orchestrate tasks.  
✅ **Custom Tools** - Plant database and companion planting tools to customize agents.  
✅ **ADK Runner** - `InMemoryRunner` for agent execution  
✅ **Session Management** - `DatabaseSessionService` for persistent sessions which can survive restarts. `EventsCompactionConfig ` to manage the history and context of a conversation within an ADK session, ensuring efficient and effective communication with agents.   
✅ **Memory Management** - `InMemoryMemoryService` for quick prototyping.  
✅ **Observability** - `LoggingPlugin`, a built-in tool for performance analysis and debugging.

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- Access to Google Generative AI API with API key

### 1. Clone this repository:

```
git clone https://github.com/AlstonChew/AI-Garden-Application.git

cd AI-Garden-Application
```

### 2. Create and activate a virtual environment (optional but recommended):

```
python -m venv venv

source venv/bin/activate # On Windows use venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add your Google API Key

- Create '.env' file
- Copy `.env.example` to `.env`
- Set your API key in '.env' file:
```
GOOGLE_API_KEY=your-api-key-here
```

Get your key: https://aistudio.google.com/app/apikey

### 5. Run the Streamlit App:

```bash
streamlit run main.py
```

### Usage
- Open the Streamlit UI in your browser (usually http://localhost:8501)
- Ask questions about plant care, gardening problems, or companion planting such as the following:
    - How do I care for tomatoes?
    - Can I plant basil with tomatoes?
    - Brown spots on tomato leaves indicate early blight.
- Interact naturally; behind the scenes the agent may call tools and provide actionable advice
