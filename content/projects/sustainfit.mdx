# SustainFit AI (AglioOlio)

SustainFit AI is an AI-powered fashion assistant that helps users measure, visualize, and alter their garments. Utilizing Gemini Vision and DALL-E 3, it provides a complete workflow from garment analysis to tailored visualization and instructions.

## 🚀 Features

- **Garment Analysis**: Automatically detect garment type and estimate flat-lay measurements using **Gemini 3 Pro Image Preview**.
- **Virtual Try-On**: Visualize how a garment looks on a person with AI-generated synthesis powered by **Gemini 3 Pro**.
- **Multilingual Tailoring Instructions**: Generate technical steps for physical alterations in multiple Southeast Asian languages (Thai, Vietnamese, Indonesian, Malay) powered by the **SEA-LION API**.
- **Robust Image Generation**: A 3-tier fallback system (Gemini 3 Pro -> Gemini 1.5 -> DALL-E) for reliable AI previews.

## 🏗️ Project Structure

- `backend/`: FastAPI server with AI service integrations.
- `frontend/`: React + Vite application with Tailwind CSS.
- `lib/`, `android/`, `ios/`, etc.: Flutter project files (Mobile integration in progress).

## 🛠️ How to Run

### 1. Backend Setup
```bash
cd backend
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

pip install -r requirements.txt
```
*Note: Ensure you create a `.env` file in the `backend` folder with your API keys.*

**Start Backend:**
```bash
uvicorn main:app --reload
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## ⚙️ Tech Stack & APIs

- **Frontend**: React, Vite, Tailwind CSS, Lucide React.
- **Backend**: FastAPI, OpenCV, Pydantic.
- **AI Integrations**:
  - **Google Gemini API**: `gemini-3-pro-image-preview` for vision analysis, try-on prompt synthesis, and image generation.
  - **SEA-LION API**: Used for specialized Southeast Asian language translation and technical tailoring instruction generation.
  - **OpenAI API**: `dall-e-3` used as a professional backup for high-quality image generation.
